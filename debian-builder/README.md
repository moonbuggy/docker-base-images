# debian-builder

This is a Debian container for building from source, for use as part of a multi-stage build.

## Installed Packages

*   autoconf
*   bison
*   build-essential
*   ca-certificates (sometimes needed to pull source from repos)
*   clang
*   flex
*   git
*   sudo

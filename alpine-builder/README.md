# alpine-builder
An Alpine container for building from source, for use as part of a multi-stage build.

## Installed Packages

*   alpine-sdk (includes build-base and more)
*   autoconf
*   bison
*   flex
*   git
*   sudo

## Build Environment
*   `APK_PROXY`     - URL for the Alpine repository to use (accepts: `http(s)://host(:port)`)
